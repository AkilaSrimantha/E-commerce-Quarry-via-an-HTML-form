E-COMMERCE EMAIL CONFIGURATION ASSIGNMENT

Task:
Perform email configuration to send query received confirmation to client and the actual query to admin once a client submits a query via an HTML form.

Solution:
This project uses EmailJS to send emails directly from an HTML form. When the client submits the query form, two emails are sent:
1. Confirmation email to the client.
2. Query details email to the admin.

Main Files:
- index.html: Contains the query form.
- style.css: Contains the design styles.
- script.js: Contains EmailJS email sending logic.
- README.md: Contains setup instructions.
